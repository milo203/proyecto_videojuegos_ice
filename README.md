# Análisis Estratégico y Predicción de Ventas de Videojuegos 🎮

## 📋 Descripción del Proyecto
Este proyecto simula el rol de un Analista de Datos para "Ice", una tienda internacional de videojuegos. El objetivo principal es analizar datos históricos de ventas globales, reseñas de usuarios y expertos, géneros y plataformas (ej. Xbox, PlayStation) para identificar patrones que determinan el éxito de un juego. 

Los *insights* generados en este análisis están diseñados para ayudar al equipo de marketing a detectar proyectos prometedores y optimizar la planificación de campañas publicitarias para el próximo año fiscal.

## 🛠️ Herramientas y Tecnologías
Para este proyecto se utilizó el siguiente entorno y librerías:
* **Entorno:** Jupyter Notebook / Visual Studio Code
* **Lenguaje:** Python
* **Manipulación y limpieza de datos:** Pandas, NumPy
* **Visualización de datos (EDA):** Matplotlib, Seaborn
* **Análisis Estadístico:** SciPy Stats, Math

## 🔍 Metodología
1. **Preprocesamiento de Datos:** Extracción, limpieza y filtrado de valores nulos o atípicos en un *dataset* masivo.
2. **Análisis Exploratorio de Datos (EDA):** Creación de gráficos para entender el ciclo de vida de las plataformas, la distribución de ventas por región (Norteamérica, Europa, Japón) y la popularidad de los géneros.
3. **Pruebas de Hipótesis:** Aplicación de estadística inferencial para validar diferencias significativas en las ventas según distintas variables (calificaciones, plataformas, géneros).

## 📈 Hallazgos Clave e Impacto de Negocio
* **Correlación de éxito:** Se demostró estadísticamente una correlación media positiva entre las puntuaciones de los críticos especializados y las ventas totales del juego, sirviendo como un indicador predictivo temprano.
* **Rentabilidad por Plataforma y Región:** Se identificaron las plataformas más prometedoras para el siguiente año, destacando claras diferencias en las preferencias de consumo dependiendo de la región geográfica.
* **El poder del Multiplataforma:** Se descubrió que la estrategia multiplataforma es un factor crucial para el volumen de ventas masivo, destacando la franquicia de **LEGO** como el caso de mayor éxito bajo este modelo.
